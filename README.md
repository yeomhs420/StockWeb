### 상세기능
 - 사용자가 종목 검색을 하면 주가 정보, 기업 정보, 재무제표를 확인할 수 있다. 
 - 검색 키워드의 공백, 대/소문자가 일치하지 않아도 검색 가능하다. 예) 삼  성전 자, S-OiL
 - 재무제표와 상위 업종/테마 메뉴에서는 pyplot그래프가 추가되어, 데이터를 시각적으로 확인할 수 있다.
 - 데이터를 가져오는 동안 로딩중임을 알리는 Spinner을 웹사이트에 표시했다.
